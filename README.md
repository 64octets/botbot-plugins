# BotBot.me Plugins

To get started:

```
$ pip install -e git+https://github.com/lincolnloop/botbotme-plugins.git#egg=botbotme-plugins
$ botbotme_shell
```

Pass a comma-separated list of modules to run a subset of the plugins:

```
$ botbotme_shell brain,images
```
